欢迎

作者：

“Cheers for a great tool that actually makes programmers want to write documentation!“

Sphinx 是一种工具，能简单方便地创建智能且美观的文档， 由 Georg Brandl 编写并在 BSD 上进行注册。

最初创建 Sphinx 的目的只是用于 Python 文档， 并且拥有优秀的机制来支持一系列语言程序的文档。 当然，本网站也是基于 reStructuredText 使用 Sphinx 来创建的。 下面列出需要着重强调的特性：

  • 格式输出： HTML (包括 Window 帮助文档)，LaTex (用于打印的 PDF 格式)， ePub, Texinfo, 用户手册，纯文本。
  • 大量引用： 语义标记和自动链接到函数、类、引用、 术语项和类似的一段信息。
  • 层级结构： 能简单的定义文档树，并能自动链接到兄弟节点、 父节点和子节点。
  • 自动目录： 普通目录和语言特有的模块模块目录
  • 源码处理： 使用 Pygments 语法高亮工具 自动对代码语法进行高亮处理。
  • 扩展： 自动检测代码片断，包含 Python 模块（API 文档） 更多
  • 贡献扩展： 在第二知识库中有50 个以上 由用户贡献 的扩展多，其中多数可用 PyPI 安装

Sphinx 使用 reStructuredText 作为其标记语言，并且其多数优势来自于 reStructuredText 的力量和坦率及其解析和翻译套件， Docutils。
